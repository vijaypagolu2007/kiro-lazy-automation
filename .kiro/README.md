\# Kiro Hooks Configuration



This project uses Kiro automation to run the desktop organizer script.



\## Hooks



\- \*\*pre-run\*\*: Validate environment or run quick checks.

\- \*\*run\*\*: Execute the main automation script  

&nbsp; Command:

&nbsp; python organize\_desktop.py --source {source} --backup {backup}



\- \*\*post-run\*\*: Generate index, logs, or upload artifacts.
